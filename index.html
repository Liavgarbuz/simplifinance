# SimpliFinance v4 A personal finance OS — budgeting, taxes, investments, net worth, life timeline, and AI chat assistant. ## 🚀 Deploy in 5 minutes (GitHub + Vercel) ### Step 1 — Push to GitHub ```bash git init git add . git commit -m "SimpliFinance v4" git remote add origin https://github.com/YOUR_USERNAME/simplifinance.git git push -u origin main ``` ### Step 2 — Deploy to Vercel 1. Go to [vercel.com](https://vercel.com) → **Add New Project** 2. Import your GitHub repo 3. Framework preset: **Vite** (auto-detected) 4. Click **Deploy** — done, you have a live URL to share ### Step 3 — (Optional) Enable cross-device cloud sync Without this step, each user's data is stored in their browser's localStorage. That works fine for personal use but data won't follow you to a new device. **Set up Supabase (free, takes ~5 minutes):** 1. Create a free project at [supabase.com](https://supabase.com) 2. Go to **SQL Editor** → paste contents of `supabase_setup.sql` → Run 3. Go to **Project Settings → API** → copy **Project URL** and **anon public** key 4. In Vercel: **Project Settings → Environment Variables** → add: - `VITE_SUPABASE_URL` = `https://xxxx.supabase.co` - `VITE_SUPABASE_ANON_KEY` = `eyJ...` 5. Redeploy (Vercel → Deployments → Redeploy) Users can now log in from any device and their data will sync. --- ## 💻 Run locally ```bash npm install npm run dev ``` Open http://localhost:5173 ```bash npm run build # production build → dist/ npm run preview # preview the build locally ``` --- ## 📁 Project structure ``` simplifinance/ ├── index.html # HTML entry point ├── vite.config.js # Vite + React config ├── vercel.json # SPA routing for Vercel ├── .env.example # Copy to .env for local cloud sync ├── supabase_setup.sql # Run once in Supabase SQL editor └── src/ ├── main.jsx # React root mount └── App.jsx # Entire app (single file) ``` --- ## ✨ Features | Tab | What it does | |-----|-------------| | **Overview** | Net worth snapshot, savings rate, 10-yr projection, AI coach | | **Budget** | Income sources, MECE expense categories, retirement contributions, tax settings | | **Taxes** | Full 2026 bracket calculation — AGI → taxable income → total tax | | **Investments** | Multi-account portfolio, live prices (Finnhub), Sharpe ratio, gains tax | | **Net Worth** | 30-year projection, debt payoff analysis, asset breakdown | | **Life Timeline** | Milestone map showing how life events impact your net worth | | **AI Chat** | Financial Q&A powered by Claude (uses your actual data as context) | --- ## 🔒 Security notes - Passwords/PINs are stored as plaintext in this version (suitable for personal use) - For production, hash PINs with bcrypt before storing — the `supabase_setup.sql` has a `pin_hash` column ready for that - All financial data is yours — no analytics, no ads, no data selling - The Anthropic API key for the AI chat is called client-side — for production, proxy it through a Vercel serverless function --- ## 🛠 Editing the app Everything is in `src/App.jsx` — one file by design.
